Spring-MVC JSON 支持

Spring MVC 提供了JSON返回结果支持功能。这个功能需要第三方组件jackson的支持。

jackson的作用是将Java对象格式化为Json风格的字符串。

使用步骤

  1. 导入 Spring-webmvc 和 jackson 包
  2. 配置Spring-webmvc
    • 在web.xml配置 DispatcherServlet
    • 添加 Spring 配置文件 spring-web.xml
      • <context:component-scan base-package="cn.tedu.web"/>
      • <mvc:annotation-driven/>
  3. 编写控制器
    • 在控制器方法上使用 @ResponseBody
    • 控制器方法的返回值就自动封装到 响应Body中返回到浏览器

案例：

pom.xml (组件坐标)

    <dependency>
        <groupId>org.springframework</groupId>
        <artifactId>spring-webmvc</artifactId>
        <version>3.2.8.RELEASE</version>
    </dependency>
    <dependency>
        <groupId>com.fasterxml.jackson.core</groupId>
        <artifactId>jackson-core</artifactId>
        <version>2.2.3</version>
    </dependency>
    <dependency>
        <groupId>com.fasterxml.jackson.core</groupId>
        <artifactId>jackson-annotations</artifactId>
        <version>2.2.3</version>
    </dependency>
    <dependency>
        <groupId>com.fasterxml.jackson.core</groupId>
        <artifactId>jackson-databind</artifactId>
        <version>2.2.3</version>
    </dependency>

web.xml Spring 控制器配置：

<servlet>
    <servlet-name>mvc</servlet-name>
    <servlet-class>
        org.springframework.web.servlet.DispatcherServlet
    </servlet-class>
    <init-param>
        <param-name>contextConfigLocation</param-name>
        <param-value>classpath:spring-web.xml</param-value>
    </init-param>
    <load-on-startup>1</load-on-startup>
</servlet>

<servlet-mapping>
    <servlet-name>mvc</servlet-name>
    <url-pattern>*.do</url-pattern>
</servlet-mapping>

spring-web.xml Spring 配置文件，打开组件扫描

 <context:component-scan 
    base-package="cn.tedu.web"/>

 <mvc:annotation-driven/>

控制器类：

package cn.tedu.web; 

@Controller
@RequestMapping("/demo")
public class JsonController {
    //@ResponseBody 来自Spring的注解，作用是
    //将控制器方法的返回结果格式化为JSON字符串
    //封装到HTTP协议的响应body中发送到客户端
    @RequestMapping("/hello.do")
    @ResponseBody
    public Object execute(){
        return "Hello World";
    }
}

测试：

请求 http://localhost:8080/SpringAjax/demo/hello.do
的JSON 返回结果。

Spring 支持的返回多种对象作为JSON

案例：

//将Map作为返回结果
@RequestMapping("/map.do")
@ResponseBody
public Object map(){
    Map<String, Object> map = 
        new HashMap<String, Object>();
    map.put("name", "Tom");
    map.put("age", 28);
    map.put("job", "抓老鼠");
    map.put("times", new int[]{1,4,6});
    return map;
}

//将List作为返回结果
@RequestMapping("list.do")
@ResponseBody
public List<String> list(){
    ArrayList<String> list =
        new ArrayList<String>();
    list.add("Tom");
    list.add("Jerry");
    list.add("Andy");
    return list;
}

//将对象作为返回结果
@RequestMapping("/object.do")
@ResponseBody
public Object object(){
    return new Person(88, "饭老师", 18);
}

测试类：

public class Person {
    private int id;
    private String name;
    private int age;
    public Person() {
    }

    public Person(int id, String name, int age) {
        super();
        this.id = id;
        this.name = name;
        this.age = age;
    }

    public int getId() {
        return id;
    }
    public void setId(int id) {
        this.id = id;
    }
    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
    public int getAge() {
        return age;
    }
    public void setAge(int age) {
        this.age = age;
    }

}

关于JSON 返回结果

  1. List 与 Array 转换为JSON以后没有差别
  2. Object 与 Map 转换为JSON以后没有差别
  3. 如果需要返回多个数据，可以将数据封装到Map或者Object中返回。
    • 不建议封装到List中返回。

提示：map作为JSON结果是非常便捷的。
